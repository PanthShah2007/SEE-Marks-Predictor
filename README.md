# 🎯 SEE Marks Predictor

A modern web-based SEE Marks & SGPA Predictor designed for Nirma University ECE students.

## 🚀 Live Demo
https://mellifluous-pegasus-81f120.netlify.app/

## 📖 About The Project

SEE Marks Predictor helps students estimate:

- Required SEE marks for a target grade
- Internal marks calculation
- Semester-wise performance analysis
- Estimated SGPA based on selected grades

The tool supports Semester 1 to Semester 4 subjects of the ECE curriculum and provides an easy-to-use futuristic interface.

## ✨ Features

- 📚 Semester-wise subject selection
- 🎯 Target grade prediction
- 📊 Internal marks calculation
- 📝 SEE marks estimation
- 🎓 SGPA estimation
- 📱 Fully responsive design
- ⚡ Fast client-side calculations
- 🌙 Modern UI/UX

## 🎯 How It Works

1. Select your semester.
2. Enter internal marks.
3. Choose your target grade.
4. View required SEE marks instantly.
5. Estimate SGPA.
